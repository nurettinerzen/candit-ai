📐 Aday Havuzu + Aday Profili Spec

Aday Havuzu (liste):
• Bu sayfa global aday havuzu / CRM — günlük operasyon yeri DEĞİL
• Tablo: Aday (link), E-posta, Telefon, Kaynak (badge), Başvuru Sayısı (yeni sütun), Kayıt Tarihi
• "Başvuru Sayısı" sütunu eklenmeli — adayın kaç ilana başvurduğunu gösterir
• Arama: ad, e-posta, telefon ile filtreleme

Aday Profili (detay):
• Aday bilgileri: Ad, E-posta, Telefon, Kaynak, Kayıt Tarihi, CV İnceleme Güveni
• CV Özeti: Kısa açıklama + güçlü yönler + görüşmede sorulması gerekenler
• Başvurular tablosu: İlan (link), Aşama (merkezi stage badge), Son Güncelleme, İşlem (başvuruya link)
• Başvuru tablosunda başvuru ID gösterilmemeli — ilan adı gösterilmeli
• "Adayı İlana Bağla" dropdown — yeni başvuru açmak için
• Stage badge'leri merkezi getRecruiterStageMeta kullanmalı
