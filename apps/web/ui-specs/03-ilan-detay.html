📐 İlan Detay / Başvuru Inbox Spec

Bu sayfa ürünün ağırlık merkezi — command center.

• Breadcrumb: "← İlan Merkezi"
• İlan meta bilgisi: durum badge + lokasyon + çalışma modeli + maaş aralığı
• KPI'lar: Toplam Başvuru, Skoru Hazır, Ort. Uyum Skoru, İşlem Bekliyor
• Pipeline filtreleri: Tümü / Başvurdu / Ön Eleme / Görüşme / İnceleme / Teklif-Sonuç (tab sayılarıyla)
• Arama + kaynak filtre + minimum skor + sıralama + CV Yükle butonu

Tablo sütunları (inbox):
• ☑ Checkbox (toplu işlem için)
• Aday: Ad, lokasyon, iletişim bilgisi (e-posta, telefon) — tek sütunda kompakt
• Kaynak: kariyer.net, manual, csv_import vb.
• Uyum Skoru: Daire içinde sayı + band etiketi (Güçlü/İyi/Orta/Zayıf)
• Eksik / Uyarı: Risk tag'leri — "3 Eksik Bilgi", "1 Uyarı" gibi kompakt etiketler
• Durum: Merkezi stage badge — tüm sayfalarla aynı (Karar Bekliyor / Bekletildi / İlerletildi / Reddedildi / Mülakat Planlandı)
• Sonraki Adım: Aksiyon linki — "Karar Ver", "İncele", "Değerlendirmede" vb.
• İşlem: Dropdown menu butonu (İlerlet, Beklet, Reddet, Görüşmeye Davet Et)

Önemli: Aday adına tıklayınca başvuru detay sayfasına gider, ayrı bir aday profil sayfasına DEĞİL.
